  • B选项：明显的错误，then要么另起一行，要么前面加;。

十一、进程的基本概念

  • int a[N] = { 2 };的意思是把a[N]第一个元素赋值为2，其他的赋值为0，也就是说，所有的元素都被赋值了，因此占空间很大。
  • int a[N];的a[N]没有赋初值，只需记录a的长度即可，占用空间很小。
