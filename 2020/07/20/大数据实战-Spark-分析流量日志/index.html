  • 虚拟机数量：1

  • Spark版本：Apache Spark 2.1.1
  • Eclipse版本：Neno.3 （4.6.3）

二、实验目的：

掌握

  • Spark原理
  • Hadoop原理
  • Shell
  • Spark Streaming
  • 二次排序
  • 序列化
  • Spark RDD

6.2需求分析

6.2.1对文本中数据记录进行排序，排序规则如下：

6.2.2具体实现步骤：

6.2.2.1按照 Serrializable 接口实现自定义排序的 Key
